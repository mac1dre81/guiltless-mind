# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-04-10

### Added
*   **Initial Release**
*   **Offline-First Editing:** Open, read, and write PDF and Markdown files entirely on-device without internet.
*   **Recent Files Persistence:** Remembering opened documents across app restarts utilizing `takePersistableUriPermission`.
*   **Document Scanner:** Advanced Google ML Kit-powered scanning.
*   **Optical Character Recognition (OCR):** Seamless extraction of text from scanned documents.
*   **Monetization Tiers:** 
    * Free (Ad-supported)
    * Premium One-time ($3.99 - Unlimited pages, Ad-removal)
    * Pro Subscription ($2.99/mo - ML Kit Scanner unlocked)
*   **UI/UX:** Fully responsive Material 3 design with Jetpack Compose, dynamic colors, and Dark Theme support.
*   **Settings Menu:** Toggles for system matching and persistent preferences stored via DataStore.
